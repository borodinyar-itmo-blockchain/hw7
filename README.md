# hw7

## How to build
```
cargo build --release
```
Then you can find the executable file in `target/release/`.

## How to run
```
./target/release/hw7 --file <FILE> --numbilets <NUMBILETS> --parameter <PARAMETER>
```

./target/release/hw7 --file ./data/1.txt --numbilets 15 --parameter 42
